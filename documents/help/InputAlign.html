The InputAlign test plugin is designed to easily set the correct input level of analogue soundcards.

To use the plugin

  1. Select an input channel.
  2. Provide the PAM with tone at a known amplitude.
  3. Enter this amplitude using the number pad.
  4. Press either "Align Selected Channels" or "Align All Channels"

The PAM will compare the amplitude it is receiving the tone with the expected amplitude and provide makeup/down gain (in software) so the two values match.
